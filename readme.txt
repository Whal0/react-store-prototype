Projekt na zaliczenie przedmiotu "Technologie Webowe" 
Informatyka Społeczna semestr 2.

Użyte narzędzia/ języki:
- HTML/CSS
- React.js
- React Router
- Material UI
- Node.js
- Commerse.js
- React-paypal-button-v2

Pisane w VSCode, użyte rozszerzenia:
- ES7 React/Redux/GraphQL/React-Native snippets
- vscode-styled-components
- npm
- Auto Rename tag
- Bracket Pair Colorizer

Projekt testowany na test buildzie za pomocą "npm start".

- Baza przedmiotow zabierana za pomoca api Commerse, 
- koszyk z możliwośćią modyfikowania ilośći przedmiotów.
- ikonie przy koszyku zmieniająca się z ilośćią przedmiotów.
- Przejscia na rózne podstrony dzięki React Router
- Inny widok koszyka w zależności czy jest pełny.
- Prosta Kasa z odpowiednimi polami, różne opcje płatnośći(Nieaktywne).
- Podsumowanie z listą i ilośćią przedmiotów wraz z łączną ceną

Komenda do zależnośći:
npm install @material-ui/core @material-ui/icons @chec/commerce.js @stripe/react-stripe-js @stripe/stripe-js react-router-dom react-hook-form react-paypal-button-v2
